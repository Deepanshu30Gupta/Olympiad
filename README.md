# Olympiad

> An AI-powered adaptive learning platform for Mathematical Olympiad preparation.

---

## Vision

Olympiad aims to transform olympiad preparation from a static collection of questions into an intelligent, personalized learning experience.

Instead of asking **"Which paper should I solve next?"**, the platform will answer:

> **"What should I learn next to maximize my improvement?"**

The long-term goal is to combine educational technology, artificial intelligence, recommendation systems, learning analytics, and software engineering into a single open-source platform that helps students prepare more effectively for mathematical olympiads.
